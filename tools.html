Überblick über meine verwendeten Tools

Windows 11

Windows ist das beliebteste Betriebssystem für den Computer. Windows wird bei der Software-Entwicklung gegenüber anderen Systemen bevorzugt. Meine Tools laufen im Browser und sind somit plattformübergreifend. Die Tipps veröffentliche ich aber für Windows 11.

GitHub

Auf GitHub veröffentliche ich aktuelle Versionen der Website und von Tools. Dort könnt ihr Verschiedenes kommentieren, wenn ihr euch dort kostenlos angemeldet habt.

Mozilla Firefox

Ein quelloffener und kostenloser Browser. Ich bin Firefox-Fan und nutze deshalb diesen Browser zur Entwicklung. Besonders fasziniert mich auch, dass er eine echte Alternative zu Chrome und den anderen Chromium-Browsers darstellt. Meine Tools werden aber auch so entwickelt, dass sie in Chrome mindestens genauso gut funktionieren wie in Firefox.

Visual Studio Code

Der beliebteste Code-Editor der Welt, welcher auch gleichzeitig quelloffen und kostenlos ist. Für die Webentwicklung ist er optimal geeignet. Verwendet habe ich ihn sowohl für die Website als auch die ganzen Tools.

Google Fonts

Für die Schriftart und auch die meisten Icons verwende ich den Dienst Google Fonts. Bei mir läuft er direkt auf dem Server, sodass die Website datenschutzfreundlich ist.

Live Server

Dies ist eine Erweiterung für Visual Studio Code. Mit ihr kann man einen lokalen Server erzeugen, auf der eine geöffnete Webseite läuft. Wenn man dort Veränderungen vornimmt, wird die Seite sofort aktualisiert, ohne dass man das manuell im Browser machen muss.
